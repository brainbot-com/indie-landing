General information

This privacy policy explains how we process personal data when you use this website.

Access data (server logs)

When you access the website, technically necessary data is processed (e.g., IP address, time of access, requested file, user agent). Processing is necessary to provide the website and ensure security.

Cookies and consent

We do not use strictly necessary cookies. Analytics cookies are only set with your consent.

Website analytics (Matomo)

We use Matomo to measure reach and improve this website. Matomo is operated by us; no data is shared with third‑party providers.

If you consent, Matomo sets cookies to recognize returning visitors. “Do Not Track” is respected. Your IP address is truncated.

In particular, we process visited pages, referrer information, and technical device/browser information.

You can change your consent anytime via Tracking settings.

Recipients and processors

We do not share data with third parties unless required by law. If service providers are used, this is done under data processing agreements according to Art. 28 GDPR.

Hosting: This website is delivered as a static site via GitHub Pages (GitHub, Inc.). The source code is hosted on GitHub, but we do not store content or user data there. When you access the site, the hosting provider processes technically necessary access data (e.g. IP address, timestamp, requested file) in server logs.

Storage period

We process personal data only as long as necessary for the stated purposes or as required by law.

Your rights

You have the right to access, rectify, delete, restrict processing, data portability, and object. You can withdraw consent at any time with effect for the future.

You also have the right to lodge a complaint with a data protection authority.

Changes to this privacy policy

We may update this privacy policy if necessary.
