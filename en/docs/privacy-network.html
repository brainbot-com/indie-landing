Privacy & Network

This page describes the default delivery state of the Indiebox and clearly separates that from intentionally enabled external features.

Default state

The box is configured so that AI usage data is not sent outward during normal use. Local processing is the default starting point.

External search is not active by default

External search services are not enabled in the delivery state. If such features are needed, they must be enabled consciously and configured explicitly.
