Usage

OpenWebUI

OpenWebUI is the primary interface for the first productive use of Indie.box. This is where users log in, test models and later work with knowledge sources and optional extensions.

Role in the system

OpenWebUI is the first place where users actually work. That is why this interface should be the first stable target after the box becomes reachable.

First login

Use the credentials from the handout. A successful login confirms that reachability, authentication and the main interface are working.

Testing models intentionally

Run the same task with at least two models. Compare quality, speed and style instead of assuming one model fits every job.
