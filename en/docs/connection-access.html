Connection & Access

This page explains the three access modes for Indie.box: fully offline directly at the box, recommended LAN access in the local network and Wi‑Fi setup by QR code.

Access modes

Offline

Works without any network by attaching screen, keyboard and mouse directly to the box.

LAN

Recommended default. Browser access through the device name from the handout.

Wi‑Fi

Simple setup path through QR code and assistant.

Recommended: LAN by Ethernet

The preferred path is Ethernet. It is the most stable way to make the box reachable on the local network and use it from laptops, tablets or phones.

Wi‑Fi by QR code

If the box should join Wi‑Fi, scan the QR code and follow setup. After that, continue in the browser.

Fully offline at the box

The Indie.box can run completely without network access. Attach screen, keyboard and mouse and work locally on the device.

If the box is not reachable

  • Check that startup is complete.
  • For LAN, verify Ethernet and local network access.
  • If the device name is not resolved, use the QR path or inspect the local network environment.
  • If no network is desired, use offline mode directly at the box.
