Knowledge Sources

Knowledge sources let the box work with local documents instead of relying only on general model knowledge.

Why this matters

This is the step where general AI usage turns into work with local material, such as notes, policies or project documents.

Adding documents sensibly

Start with a small and clearly scoped set of documents. That makes it much easier to verify whether the system is actually using the content in a useful way.

What to expect

  • Answers should move closer to local material.
  • Clear document structure improves results.
  • Unstructured uploads usually make evaluation harder.
